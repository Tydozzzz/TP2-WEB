1. Qu'est-ce que l'HTML ?

HTML signifie HyperText Markup Language. C'est le langage utilisé pour créer la structure des pages web. Il permet d'organiser le contenu : titres, paragraphes, images, liens, listes, etc.

2. Structure de base d'une page HTML

Voici la structure minimale d’un document HTML :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
</head>
<body>
    Contenu de la page
</body>
</html>
  • <!DOCTYPE html> : indique qu’on utilise HTML5
  • <html> : racine du document
  • <head> : informations pour le navigateur
  • <body> : contenu visible par l’utilisateur

3. Les balises HTML

L’HTML fonctionne avec des balises. La plupart ont une balise ouvrante et une balise fermante : 

<p>Ceci est un paragraphe</p>

Certaines balises sont dites "auto-fermantes" : 

<img src="image.jpg" alt="Une image">

4. Le texte : titres et paragraphes

Pour écrire du texte :

  • <h1> à <h6> : titres (du plus important au moins important)
  • <p> : paragraphe
<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<p>Ceci est un paragraphe.</p>

5. Les liens

Les liens permettent de naviguer entre les pages grâce à la balise <a>. 

<a href="https://www.google.com">Aller sur Google</a>

6. Les images

Pour afficher une image, on utilise la balise <img>. 

<img src="chat.jpg" alt="Un chat mignon">
  • src : chemin de l’image
  • alt : texte alternatif (important pour l’accessibilité)

7. Les listes

Il existe deux types de listes :

Liste non ordonnée (à puces)

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Liste ordonnée (numérotée)

<ol>
    <li>Installer un éditeur</li>
    <li>Écrire le code</li>
    <li>Ouvrir dans le navigateur</li>
</ol>

8. Conclusion

L’HTML sert à structurer le contenu d’un site web. Il est souvent utilisé avec :

  • CSS pour le style (couleurs, tailles, mise en page)
  • JavaScript pour l’interactivité

Une fois les bases maîtrisées, tu pourras créer tes propres pages web.
